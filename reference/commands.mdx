## Main Command

flowState is a single binary with no subcommands. The TUI launches immediately when you run the executable.

<CodeGroup>
```bash Linux/macOS
./flowstate
```

```bash Windows
.\flowstate.exe
```

```bash npm (all platforms)
flowstate
```

```bash Go Install
go install github.com/Jericoz-JC/flowState-CLI/cmd/flowState@latest
```
</CodeGroup>

### First Run

On first run, flowState will:

1. Initialize SQLite database at `~/.config/flowState/flowState.db`
2. Download the embedding model (~90MB) to `~/.config/flowState/models/`
3. Launch the TUI interface with home screen

<Info>
The application logs debug output to `debug.log` in the working directory.
</Info>

## Navigation Shortcuts

flowState uses keyboard shortcuts to navigate between screens. All shortcuts work globally from any screen.

### Global Navigation

<ParamField path="Ctrl+X" type="shortcut">
  **Quick Capture** - Open quick note capture modal from anywhere
</ParamField>

<ParamField path="Ctrl+N" type="shortcut">
  **Notes Screen** - View and manage your notes
</ParamField>

<ParamField path="Ctrl+T" type="shortcut">
  **Todos Screen** - View and manage your tasks
</ParamField>

<ParamField path="Ctrl+F" type="shortcut">
  **Focus Screen** - Start Pomodoro-style focus sessions
</ParamField>

<ParamField path="Ctrl+/" type="shortcut">
  **Search Screen** - Semantic search across all notes
</ParamField>

<ParamField path="Ctrl+G" type="shortcut">
  **Mind Map** - Visual graph of notes and connections
</ParamField>

<ParamField path="Ctrl+L" type="shortcut">
  **Link Modal** - Create/view links for selected note or todo
</ParamField>

<ParamField path="Ctrl+H" type="shortcut">
  **Home Screen** - Return to home/help screen (highest priority navigation)
</ParamField>

<ParamField path="?" type="shortcut">
  **Help Modal** - Context-sensitive help for current screen
</ParamField>

<ParamField path="Esc" type="shortcut">
  **Go Back / Cancel** - Close modals, cancel actions, exit edit mode
</ParamField>

<ParamField path="q" type="shortcut">
  **Quit Application** - Exit flowState (also Ctrl+C)
</ParamField>

<Note>
On macOS, `Ctrl` is used (not `Cmd`). This ensures cross-platform consistency.
</Note>

## Configuration

flowState stores its configuration and data in `~/.config/flowState/`:

<ResponseField name="config.json" type="file">
  Configuration file with database path, model path, and settings
</ResponseField>

<ResponseField name="flowState.db" type="file">
  SQLite database containing all notes, todos, sessions, and links
</ResponseField>

<ResponseField name="models/" type="directory">
  Directory containing the embedding model (all-MiniLM-L6-v2, ~90MB)
</ResponseField>

<ResponseField name="debug.log" type="file">
  Debug log file created in working directory (not config directory)
</ResponseField>

<Tip>
The database uses SQLite's `PRAGMA foreign_keys = ON` for referential integrity. All data is stored locally with no cloud dependencies.
</Tip>
