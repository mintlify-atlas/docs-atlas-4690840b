## Global Navigation

These shortcuts work from **any screen** in flowState CLI:

| Key | Action | Description |
|-----|--------|-------------|
| `Ctrl+X` | Quick Capture | Instantly capture a note from anywhere |
| `Ctrl+N` | Notes Screen | Navigate to Notes |
| `Ctrl+T` | Todos Screen | Navigate to Todos |
| `Ctrl+F` | Focus Screen | Navigate to Focus Sessions |
| `Ctrl+/` | Search Screen | Navigate to Semantic Search |
| `Ctrl+G` | Mind Map Screen | Navigate to Mind Map |
| `Ctrl+L` | Link Modal | Open linking modal for selected item |
| `Ctrl+H` | Home Screen | Return to home/help screen |
| `?` | Help Modal | Show context-sensitive help |
| `Esc` | Go Back / Cancel | Exit current mode or modal |
| `q` | Quit | Exit flowState CLI |

<Note>
  **macOS users**: You can use `Cmd` (вҢҳ) instead of `Ctrl` for all shortcuts.
</Note>

## Notes Screen

### List View

| Key | Action | Description |
|-----|--------|-------------|
| `c` | Create | Create new note |
| `e` | Edit | Edit selected note |
| `p` | Preview | Preview note (read-only markdown) |
| `d` | Delete | Delete selected note (with confirmation) |
| `/` | Filter | Open search filter |
| `s` | Sort | Cycle sort mode (DateвҶ“ вҶ’ Title вҶ’ DateвҶ‘) |
| `t` | Tag Filter | Filter by tags (multi-select picker) |
| `Ctrl+R` | Reset | Reset all filters |
| `Ctrl+L` | Links | Open linking modal |
| `j` / `вҶ“` | Down | Move selection down |
| `k` / `вҶ‘` | Up | Move selection up |
| `Ctrl+H` | Home | Return to home screen |

### Edit Mode

When creating or editing a note:

| Key | Action | Description |
|-----|--------|-------------|
| `Tab` | Switch Field | Toggle between title and body |
| `Enter` | Save / New Line | Save (in title) or new line (in body) |
| `Ctrl+S` | Save | Save note and return to list |
| `Ctrl+E` | Preview | Toggle markdown preview |
| `Ctrl+B` | Bold | Bold selected text |
| `Ctrl+I` | Italic | Italic selected text |
| `Ctrl+G` | Quick-Tag | Open tag picker (add existing tags) |
| `Esc` | Cancel | Cancel and return to list |

### Preview Mode

| Key | Action |
|-----|--------|
| `e` | Edit | Switch to edit mode |
| `p` | Close | Close preview |
| `Esc` | Close | Close preview |

### Tag Picker (Ctrl+G in Edit Mode)

| Key | Action |
|-----|--------|

| `Space` | Select | Toggle tag selection |

## Common Patterns

### Navigation

flowState CLI uses vim-style navigation wherever possible:

- `j` / `вҶ“` - Move down
- `k` / `вҶ‘` - Move up
- `h` - Move left (in Mind Map)
- `l` - Move right (in Mind Map)

### Input Fields

- `Tab` - Switch between fields
- `Shift+Tab` - Switch to previous field
- `Ctrl+S` - Save
- `Esc` - Cancel

### Filters and Pickers

- `вҶ‘` / `вҶ“` - Navigate options
- `Space` - Toggle selection (multi-select)
- `Enter` - Confirm selection
- `Esc` - Cancel without changes

### Modifiers (Edit Mode)

- `Ctrl+B` - Bold text
- `Ctrl+I` - Italic text
- `Ctrl+E` - Toggle preview
- `Ctrl+G` - Quick-Tag picker

## Platform Differences

### macOS

On macOS, you can use either `Ctrl` or `Cmd` (вҢҳ) for all modifier shortcuts:

- `Ctrl+N` or `Cmd+N` - Notes
- `Ctrl+S` or `Cmd+S` - Save
- etc.

<Warning>
  **Note**: `Cmd+H` may hide windows on macOS. Use `Ctrl+H` to return to home screen.
</Warning>

### Windows/Linux

Use `Ctrl` for all modifier shortcuts.
