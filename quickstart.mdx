## Welcome to flowState CLI!

This guide will walk you through creating your first note, managing todos, and starting a focus session. By the end, you'll understand the core workflows that make flowState CLI a powerful productivity tool.

<Note>
  **First launch**: On first run, flowState will initialize the database and download the embedding model (~90MB). This happens only once.
</Note>

## Launch flowState

After [installation](/installation), launch flowState CLI:

```bash
flowstate
```

You'll see the home screen with navigation options:

- `Ctrl+N` - Notes
- `Ctrl+T` - Todos
- `Ctrl+F` - Focus Sessions
- `Ctrl+/` - Semantic Search
- `Ctrl+G` - Mind Map
- `Ctrl+X` - Quick Capture (works from anywhere!)

<Tip>
  Press `?` on any screen to see context-sensitive help with all available keyboard shortcuts.
</Tip>

## Creating Notes

<Steps>
  <Step title="Navigate to Notes">
    Press `Ctrl+N` to open the Notes screen.
  </Step>
  
  <Step title="Create a new note">
    Press `c` to create a new note. You'll see:
    - **Title field** (focused by default)
    - **Body field** (press Tab to switch)
  </Step>
  
  <Step title="Write your note">
    ```markdown
    Title: Project Ideas
    Body: 
    Need to research #ai tools for productivity
    See also [[Focus Techniques]] for time management
    ```
    
    <Note>
      - `#hashtag` - Auto-extracted as tags
      - `@mention` - Also works as tags
      - `[[Wikilink]]` - Creates relationships between notes
      - Press `Enter` in body to create new lines
    </Note>
  </Step>
  
  <Step title="Save the note">
    Press `Ctrl+S` to save, or `Esc` to cancel.
    
    Your note appears in the list with:
    - Creation date
    - Title
    - Tags as colored badges
  </Step>
  
  <Step title="Preview your note">
    Select a note and press `p` to preview it in read-only mode.
    
    Press `e` from preview to edit, or `Esc` to close.
  </Step>
</Steps>

## Managing Todos

<Steps>
  <Step title="Navigate to Todos">
    Press `Ctrl+T` to open the Todos screen.
  </Step>
  
  <Step title="Create a todo">
    Press `c` to create a new todo. You'll see:
    - **Title field** (required)
    - **Description field** (optional)
    - **Priority dropdown** (Low, Medium, High)
    - **Due date field** (optional)
  </Step>
  
  <Step title="Fill in todo details">
    ```
    Title: Review documentation #urgent
    Description: Check all code examples and API references
    Priority: High
    Due Date: 2026-03-15
    ```
    
    <Tip>
      Use `Tab` to move between fields. Tags (`#hashtag`) work in both title and description.
    </Tip>
  </Step>
  
  <Step title="Save and manage">
    - Press `Ctrl+S` to save
    - Press `Space` on a todo to toggle completion
    - Press `e` to edit an existing todo
    - Press `v` to preview full todo details
  </Step>
</Steps>

## Focus Sessions

Use the Pomodoro-style timer to stay focused.

<Steps>
  <Step title="Navigate to Focus">
    Press `Ctrl+F` to open the Focus Sessions screen.
  </Step>
  
  <Step title="(Optional) Set duration">
    Press `d` to open the duration picker:
    - Use `←/→` arrows to select duration (auto-saves)
    - Press `Tab` to switch between work/break duration
    - Press `Enter` or wait 500ms to auto-exit
    
    **Default**: 25 min work / 5 min break
  </Step>
  
  <Step title="Start the timer">
    Press `s` to start the focus session.
    
    You'll see:
    - Large ASCII timer countdown
    - Progress ring with gradient fill
    - Current session count (● completed, ○ remaining)
    - Mode indicator (🍅 Work Session)
  </Step>
  
  <Step title="Manage the session">
    - `p` - Pause the timer
    - `s` - Resume (when paused)
    - `b` - Skip to break / Skip break
    - `c` - Cancel session
  </Step>
  
  <Step title="View history and stats">
    Press `h` to view:
    - Session history (all completed sessions)
    - Total focus time
    - Current streak (consecutive focus days)
    - 7-day activity chart
  </Step>
</Steps>

## Semantic Search

Find notes by meaning, not just keywords.

<Steps>
  <Step title="Navigate to Search">
    Press `Ctrl+/` to open Semantic Search.
  </Step>
  
  <Step title="Enter a query">
    Type a natural language query:
    ```
    productivity techniques
    ```
    
    Press `Enter` to search.
  </Step>
  
  <Step title="Browse results">
    Results are ranked by semantic similarity:
    - Use `j/k` or arrow keys to navigate
    - Press `Enter` to open a note
    - Press `Esc` to edit query
  </Step>
</Steps>

<Note>
  **How it works**: flowState uses local ONNX embeddings to find notes with similar meaning. The model (~90MB) is downloaded on first run and stored locally - no cloud dependencies.
</Note>

## Linking Notes & Todos

Create relationships between items.

<Steps>
  <Step title="Select an item">
    Select a note or todo in the list view.
  </Step>
  
  <Step title="Open linking modal">
    Press `Ctrl+L` to open the Links modal.
  </Step>
  
  <Step title="Create a link">
    Press `c` to create a new link:
    1. Select link type (related, contains, references)
    2. Select target item (note or todo)
    3. Press `Enter` to confirm
  </Step>
  
  <Step title="View linked items">
    All linked items appear in the Links modal. Press `Enter` to navigate to a linked item.
  </Step>
</Steps>

## Mind Map

Visualize your note connections.

<Steps>
  <Step title="Navigate to Mind Map">
    Press `Ctrl+G` to open the Mind Map screen.
  </Step>
  
  <Step title="Navigate the graph">
    - `h/j/k/l` - Move between nodes (vim-style)
    - `+/-` - Zoom in/out
    - `Enter` - Open selected note
  </Step>
  
  <Step title="View help">
    Press `?` to see all available keyboard shortcuts.
  </Step>
</Steps>

## Global Shortcuts

These work from **any screen**:

| Shortcut | Action |
|----------|--------|
| `Ctrl+X` | Quick Capture (instant note) |
| `Ctrl+N` | Navigate to Notes |
| `Ctrl+T` | Navigate to Todos |
| `Ctrl+F` | Navigate to Focus Sessions |
| `Ctrl+/` | Navigate to Semantic Search |
| `Ctrl+G` | Navigate to Mind Map |
| `Ctrl+L` | Open Links modal |
| `Ctrl+H` | Return to Home |
| `?` | Show help for current screen |
| `q` | Quit application |

<Tip>
  On macOS, you can use `Cmd` instead of `Ctrl` for all shortcuts.
</Tip>
